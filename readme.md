Sentiment Analysis for Buyer & Seller Journey

=====================Project Overview========================================
This project focuses on analyzing customer reviews to understand the 
sentiment expressed by buyers and sellers throughout the e-commerce journey. 
Using Natural Language Processing (NLP) and Machine Learning, the system 
classifies reviews into **Positive**, **Neutral**, or **Negative** sentiments. 
The project also includes exploratory data analysis (EDA), interactive 
dashboards, and business insights to help identify customer satisfaction 
trends and areas for improvement.

=========================Objectives===========================================
- Analyze customer reviews using sentiment analysis.
- Classify reviews into Positive, Neutral, and Negative categories.
- Identify trends in customer feedback.
- Visualize insights through an interactive dashboard.
- Provide recommendations based on customer sentiments.



=========================Features=============================================
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- TF-IDF Vectorization
- Logistic Regression Model
- Model Evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion 
    Matrix
- Custom Review Sentiment Prediction
- Interactive Tableau Dashboard
- Business Insights & Recommendations


=================================Dataset======================================
The project uses an Amazon Product Reviews dataset containing customer 
reviews, ratings, product information, and recommendation data.


============================Project Workflow==================================

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering using TF-IDF
4. Train-Test Split
5. Model Training (Logistic Regression)
6. Model Evaluation
7. Sentiment Prediction
8. Exploratory Data Analysis
9. Dashboard Creation
10. Report & Presentation
